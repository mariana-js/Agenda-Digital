📘 Sobre o Sistema

Nome do Sistema: Agenda Digital Empresarial

Versão: 1.0.0

Atualizado em: 21/06/2025

🔍 Descrição

A Agenda Digital Empresarial é uma aplicação web desenvolvida para facilitar o acesso aos contatos de funcionários dentro de um ambiente organizacional.

🚀 Principais Funcionalidades

  • Consulta de contatos por nome;
  • Exibição dos aniversariantes do mês;
  • Visualização de dados como e-mail, telefones e ramal;
  • Login para acessar módulo adminstrativo;
  • Área administrativa com permissões para cadastrar, editar e excluir contatos, setor, ramal e usuário;
  • Controle de visualização de dados confidenciais.

⚙️ Tecnologias Utilizadas

  • Angular (Frontend)
  • Spring Boot (Backend)
  • PostgreSQL (Banco de Dados)
  • API do IBGE (Estados e Cidades)
